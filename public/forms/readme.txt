RTA Forms Directory
==================

This directory contains all the RTA forms that users can download from the website.

To add new forms:
1. Place PDF files in this directory
2. Use descriptive filenames (e.g., "isin_application_form.pdf")
3. Update the forms list in the RTA Forms page if needed

Current forms should include:
- ISIN Application Form
- Company Details Form
- Auditor Certificate Format
- Board Resolution Format
- Shareholder Registration Form
- Address Change Request
- Nomination Form
- Transmission Form
- And other RTA-related forms

Note: Make sure all PDF files are properly named and accessible.
